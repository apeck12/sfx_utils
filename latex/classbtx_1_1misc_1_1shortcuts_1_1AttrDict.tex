\doxysubsection{Detailed Description}
Nested Attribute Dictionary. 

A class to convert a nested Dictionary into an object with key-\/values accessible using attribute notation (Attr\+Dict.\+attribute) in addition to key notation (Dict\mbox{[}\char`\"{}key\char`\"{}\mbox{]}). This class recursively sets Dicts to objects, allowing you to recurse into nested dicts (like\+: Attr\+Dict.\+attr.\+attr)
